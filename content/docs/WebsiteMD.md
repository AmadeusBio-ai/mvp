# AmadeusBio.ai

> **The Agentic Plugin for Bioinformatics Analysis**

AmadeusBio.ai is a specialized plugin designed for agentic systems (such as Gemini CLI). It provides the necessary instructions, protocols, and interface capabilities for AI agents to perform complex bioinformatic analysis autonomously.

## Key Features

- **Natural Language Tooling**: Control local bioinformatics software like **PyMOL** using simple natural language commands.
- **Data Retrieval**: Seamlessly fetch sequences, protein structures, and metadata from major biological databases including **UniProt**, **PDB**, and **EMBL-EBI**.
- **Agentic Power**: Leverages the full potential of modern coding agents for script generation, data processing, and automated research workflows.
